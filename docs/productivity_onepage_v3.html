⌨️ CLI 接口

简单命令，强大功能

deep lux --cli
$ ./DeepLux create-project CircleDetect
✓ Created: CircleDetect.dproj

$ ./DeepLux add-module CircleDetect.dproj GrabImage --name "采集"
$ ./DeepLux add-module CircleDetect.dproj FindCircle --name "检测"
$ ./DeepLux add-module CircleDetect.dproj ShowImage --name "显示"
✓ Module added: GrabImage, FindCircle, ShowImage

$ ./DeepLux connect CircleDetect.dproj <id1> <id2>
$ ./DeepLux connect CircleDetect.dproj <id2> <id3>
✓ Connected: 采集 → 检测 → 显示

$ ./DeepLux run CircleDetect.dproj
▶ Running flow...
✓ 执行完成，检测到 3 个圆
